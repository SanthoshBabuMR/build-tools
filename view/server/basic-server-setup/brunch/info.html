README

Unlike grunt/gulp, brunch makes it mandatory to start a brunch application with a boilerplate; This is what is called as 'skeleton' in brunch.

Use brunch new <skeleton-path> to download a skeleton into local project repo.

List of skeletons available for brunch can be found at http://brunch.io/skeletons.html

Dependencies
Dead-simple brunch skeleton
brunch new gh:brunch/dead-simple
