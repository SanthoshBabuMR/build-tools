CORS server setup using grunt

try gulp try brunch Home
Dependencies
grunt
npm install --save-dev grunt
grunt-contrib-connect
npm install --save-dev grunt-contrib-connect
Testing/usage

To test a CORS server, include any asset from CORS server in a web page and open that page from a different server.

Eg: Place the snippet of code, <img src="http://localhost:43241/public/asset/logo.png" alt="stumble upon logo" /> in a web page and launch the web page using a different server.

Starting task

grunt startCORSServer
